|***********************************************************************|
|				README / Galería de Blizzard							|
|***********************************************************************|
| Este proyecto es el tomar la galería actual de blizzard y emularla	|
| de una forma sencilla y adaptable. Además de URL's amigables (SEO),	|
| fácil control en ella, segura, eficaz, editable, configurable			|
| y multi-lenguajes.													|
|***********************************************************************|
|				Forma de instalación / Galería de Blizzard				|
|***********************************************************************|
| 1.-Ejecutar la SQL incluida en la carpeta 'sql'.						|
| 2.-Abrir el archivo: config.main.php encontrado en la ruta:			|
|	 system/configs/config.main.php										|
| 3.-Configurar lo ahí indicado											|
| 4.-Dar permisos de lectura y escritura a la carpeta 'uploads'			|
| 5.-Crear una cuenta, ir a la base de datos y marcarla en la sección   |
| 	 Group como '2'. 													|
| 6.-Disfrutar la Galería												|
|***********************************************************************|
| 				Sistema de Errores / Galería de Blizzard 				|
|***********************************************************************|
| Definición: Error 403													|
| Descripción: Error de acceso a una carpeta o archivo					|
| Definición: Error 105													|
| Descripción: Error por falta de un archivo							|
| Definición: Error 404													|
| Descripción: URL no encontrada										|
